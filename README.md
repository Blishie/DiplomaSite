# DiplomaSite — Aval Emporium

Дипломный проект, разработанный во время обучения в академии ИМСИТ. Представляет собой [Многостраничный веб-сайт для заказа услуг с интерактивными элементами].

## 🛠 Стек технологий
* **Frontend:** HTML5, CSS3 (адаптивная верстка), JavaScript (ES6+).
* **Инструменты:** VS Code, Figma (верстка по макетам).

## 🚀 Реализованный функционал
* Полностью адаптивная верстка (сайт корректно отображается на смартфонах, планшетах и ПК).
* Валидация форм обратной связи на JavaScript.
* Интерактивные слайды / Анимации элементов.
* Чистая и понятная структура проекта для поддержки кода.

## 📁 Структура проекта
* `/pages` — дополнительные страницы сайта.
* `/css` — стили проекта.
* `/js` — скрипты и логика на JavaScript.
* `/images` — оптимизированные графические ресурсы.
